# 智慧財產權
```

https://www.tipo.gov.tw/tw/mp-1.html
```
```
智慧財產權（英語：intellectual property），是人類智慧創造出來的無形的財產，主要涉及著作權、專利、商標等領域。
音樂和文學等形式的藝術作品，以及一些發現、發明、詞語、詞組、符號、設計都能被當作智慧財產而受到保護。

https://en.wikipedia.org/wiki/Intellectual_property
```
